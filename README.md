# repo-wiki

Local-first repository wiki generator MVP.

## Commands
- `python -m repo_wiki.main init`
- `python -m repo_wiki.main index`
- `python -m repo_wiki.main update`
- `python -m repo_wiki.main search "<query>"`
- `python -m repo_wiki.main graph <module>`
- `python -m repo_wiki.main sync`
- `python -m repo_wiki.main verify --ci`
- `python -m repo_wiki.main cost-estimate`
